# Документация

**VP 3DViewer (OpenGL)** — программа для визуализации трёхмерных моделей с использованием OpenGL.

Разработана на языке C++20 с использованием GUI-библиотеки Qt 6.9.0, QOpenGLWidget, GLSL 330 и архитектуры MVC.

Сборка программы настроена с помощью Makefile со стандартным набором целей для GNU-программ: all, install, uninstall, clean, dvi, dist, tests.

## Возможности

1. Загрузка каркасной модели из файла формата OBJ (поддержка списка вершин и поверхностей);
2. Вращение модели левой кнопкой мыши, сдвиг правой кнопкой, зум колесом мыши;
3. Перемещение, поворот и масштабирование модели через панель управления;
4. Тип проекции: параллельная и центральная.

## Визуализация (OpenGL)

- Рендеринг через QOpenGLWidget + GLSL 330 шейдеры;
- Трансформации накапливаются в матрице на CPU, вершины не модифицируются — нулевая нагрузка на CPU каждый кадр;
- Вершины загружаются на GPU один раз (VAO/VBO/EBO);
- **ShadeMode**: заливка треугольников с Blinn-Phong освещением (ambient + diffuse + specular) через screen-space derivatives;
- **Ребра**: сплошная, пунктирная, **отсутствует**;
- **Вершины**: круг, квадрат, **отсутствует**;

## Управление

| Действие | Управление |
|----------|------------|
| Вращение | ЛКМ + движение |
| Сдвиг | ПКМ + движение |
| Зум | Колесо мыши |
| Сброс вида | Кнопка «Сброс» |

## Прочее

- Настройки (цвет фона, тип/цвет/размер вершин и рёбер, проекция, ShadeMode) сохраняются между перезапусками программы;
- Счётчик FPS (обновляется в реальном времени);
- Сохранение скриншотов в BMP/JPEG;
- Запись скринкастов в GIF (640×480, 10 FPS, 10 секунд);

*Для работы с GIF требуется ImageMagick:*

`sudo apt install imagemagick`
